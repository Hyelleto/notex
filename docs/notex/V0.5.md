## TDD 流程

### 写集成测试

```rust
#[test]
fn us15_switch_theme() {
    let mut config = AppConfig::default();
    config.theme = "dark".into();
    assert_eq!(config.theme, "dark");
}

#[test]
fn us16_persist_config() {
    let store = Store::new("test_config.json");
    store.set("theme", "dark");
    store.save().unwrap();
    let loaded = Store::new("test_config.json");
    assert_eq!(loaded.get("theme"), Some("dark".into()));
    std::fs::remove_file("test_config.json").unwrap();
}

#[test]
fn us17_font_size() {
    let mut config = AppConfig::default();
    config.font_size = 18;
    assert_eq!(config.font_size, 18);
}
```

### 写实现

- Rust 侧：`AppConfig` 结构体 + `tauri-plugin-store` 集成
- 前端：`Store.load()` / `store.set()` / `store.save()`
