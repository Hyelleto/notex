# V1.1 — 自定义插件（笔记加密）⭐ TDD

**目标**：开发一个加密插件，对笔记内容加密存储。

---

## 用户故事

| ID | 用户故事 | 验收标准 |
|----|----------|----------|
| US32 | 我想加密保存笔记 | 加密保存 → 文件内容不可读 |
| US33 | 我想解密读取笔记 | 输入密钥 → 内容正确显示 |

---

## TDD 流程

### 写集成测试

```rust
#[test]
fn us32_encrypt_note() {
    let key = generate_key();
    let encrypted = encrypt("Hello", &key).unwrap();
    assert_ne!(encrypted, "Hello");
}

#[test]
fn us33_decrypt_note() {
    let key = generate_key();
    let encrypted = encrypt("Hello", &key).unwrap();
    let decrypted = decrypt(&encrypted, &key).unwrap();
    assert_eq!(decrypted, "Hello");
}

#[test]
fn us33_wrong_key_fails() {
    let key = generate_key();
    let wrong_key = generate_key();
    let encrypted = encrypt("Hello", &key).unwrap();
    assert!(decrypt(&encrypted, &wrong_key).is_err());
}
```

---

## 实现要点

- 自定义插件结构：`Builder::new("crypto")`
- 加密算法：AES-256-GCM
- 插件权限定义：`permissions/` 目录
