
## 第 3 步：了解项目结构

Tauri 项目由**前端**和 **Rust 后端**两部分组成：

```
notex/
├── index.html              ← 前端入口（Vite 要求在根目录）
├── package.json            ← npm 配置
├── tsconfig.json           ← TypeScript 配置
├── vite.config.ts          ← Vite 构建配置
├── src/                    ← 前端代码
│   ├── main.ts             ← 前端入口脚本
│   └── styles.css          ← 样式
└── src-tauri/              ← Rust 后端
    ├── build.rs            ← Tauri 构建脚本
    ├── Cargo.toml          ← Rust 依赖
    ├── tauri.conf.json     ← Tauri 应用配置
    ├── capabilities/       ← 权限声明
    │   └── default.json
    ├── icons/              ← 应用图标
    └── src/
        ├── main.rs         ← Rust 二进制入口
        └── lib.rs          ← Tauri 配置入口
```

**前端与后端的关系**：

```
前端（WebView）  ─── invoke() ──→  Rust 后端（系统进程）
   UI 渲染         IPC 通信          系统调用 + 安全控制
```

- 前端运行在 WebView 中，负责 UI 渲染
- Rust 运行在系统进程中，负责文件/数据库/系统 API
- 通过 `invoke()` 通信（前端调 Rust 命令）

## 第 4 步：创建配置文件

**package.json**：
```json
{
  "name": "notex",
  "private": true,
  "version": "0.1.0",
  "type": "module",
  "scripts": { "dev": "vite", "build": "tsc && vite build", "tauri": "tauri" },
  "devDependencies": { "@tauri-apps/cli": "^2", "typescript": "^5", "vite": "^6" },
  "dependencies": {}
}
```

**tsconfig.json**：
```json
{
  "compilerOptions": {
    "target": "ES2021", "module": "ESNext", "moduleResolution": "bundler",
    "strict": true, "noEmit": true, "isolatedModules": true, "skipLibCheck": true
  },
  "include": ["src"]
}
```

**vite.config.ts**：
```ts
import { defineConfig } from "vite";
export default defineConfig({
  clearScreen: false,
  server: { port: 1420, strictPort: true, watch: { ignored: ["**/src-tauri/**"] } },
});
```

## 第 6 步：创建 Rust 后端

**src-tauri/Cargo.toml**：
```toml
[package]
name = "notex"
version = "0.1.0"
description = "NoteX - A simple note-taking app"
edition = "2021"

[lib]
name = "notex_lib"
crate-type = ["staticlib", "cdylib", "rlib"]

[build-dependencies]
tauri-build = { version = "2", features = [] }

[dependencies]
tauri = { version = "2", features = [] }
```

**src-tauri/build.rs**：
```rust
fn main() { tauri_build::build() }
```

**src-tauri/src/main.rs**：
```rust
#![cfg_attr(not(debug_assertions), windows_subsystem = "windows")]
fn main() { notex_lib::run(); }
```

**src-tauri/src/lib.rs**：
```rust
use tauri::Builder;
pub fn run() {
    Builder::default().run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

**src-tauri/tauri.conf.json**：
```json
{
  "productName": "NoteX",
  "version": "0.1.0",
  "identifier": "com.notex.app",
  "build": {
    "frontendDist": "../dist",
    "devUrl": "http://localhost:1420",
    "beforeDevCommand": "npm run dev",
    "beforeBuildCommand": "npm run build"
  },
  "app": {
    "windows": [{ "label": "main", "title": "NoteX", "width": 1024, "height": 768, "center": true, "resizable": true }],
    "security": { "csp": null }
  },
  "bundle": { "active": true, "targets": "all", "icon": ["icons/32x32.png", "icons/128x128.png", "icons/icon.ico"] }
}
```

**src-tauri/capabilities/default.json**：
```json
{
  "$schema": "../gen/schemas/desktop-schema.json",
  "identifier": "default",
  "description": "NoteX 默认权限",
  "windows": ["main"],
  "permissions": ["core:default"]
}
```

