## TDD 流程

### 写集成测试

```rust
#[test]
fn us34_search_finds_keyword() {
    let notes = vec![
        Note { id: 1, title: "Tauri".into(), content: "Tauri 是一个框架".into() },
        Note { id: 2, title: "Rust".into(), content: "Rust 很强大".into() },
    ];
    let results = search_notes(&notes, "Tauri");
    assert_eq!(results.len(), 1);
    assert_eq!(results[0].id, 1);
}

#[test]
fn us34_search_case_insensitive() {
    let notes = vec![Note { id: 1, title: "t".into(), content: "Hello World".into() }];
    let results = search_notes(&notes, "hello");
    assert_eq!(results.len(), 1);
}

#[test]
fn us34_search_no_match() {
    let notes = vec![Note { id: 1, title: "t".into(), content: "Hello".into() }];
    let results = search_notes(&notes, "xyz");
    assert_eq!(results.len(), 0);
}
```

## 实现要点

- `emit`：搜索进度事件
- 前端 `listen`：实时更新搜索结果
