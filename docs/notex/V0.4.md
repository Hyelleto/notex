# V0.4 — 笔记存本地文件 ⭐ TDD

**目标**：笔记持久化到本地文件系统，支持打开/保存文件。

**TDD**：✅ 用户故事 → 集成测试 → 实现。

## 用户故事

| ID | 用户故事 | 验收标准 |
|----|----------|----------|
| US11 | 我想把笔记保存为文件 | 点击保存 → 选择路径 → 文件写入成功 |
| US12 | 我想打开本地 .md 文件 | 点击打开 → 选择文件 → 内容显示在编辑区 |

## TDD 流程

### 写集成测试

```rust
// US11：保存文件
#[test]
fn us11_save_to_file() {
    let path = "test_save.md";
    cmd_save_to_file(path, "# Hello".into()).unwrap();
    let content = std::fs::read_to_string(path).unwrap();
    assert_eq!(content, "# Hello");
    std::fs::remove_file(path).unwrap();
}

// US12：打开文件
#[test]
fn us12_open_file() {
    std::fs::write("test_open.md", "# World").unwrap();
    let content = cmd_load_from_file("test_open.md".into()).unwrap();
    assert_eq!(content, "# World");
    std::fs::remove_file("test_open.md").unwrap();
}

// US13：保存到新路径
#[test]
fn us13_save_new_path() {
    let path = "test_new_dir/new_file.md";
    cmd_save_to_file(path, "content".into()).unwrap();
    assert!(std::path::Path::new(path).exists());
    std::fs::remove_file(path).unwrap();
    std::fs::remove_dir("test_new_dir").unwrap();
}

// US14：打开不存在的文件
#[test]
fn us14_open_nonexistent() {
    let result = cmd_load_from_file("nonexistent.md".into());
    assert!(result.is_err());
}
```

### 写实现

```rust
#[tauri::command]
async fn save_to_file(path: String, content: String) -> Result<(), String> {
    if let Some(parent) = std::path::Path::new(&path).parent() {
        std::fs::create_dir_all(parent).map_err(|e| e.to_string())?;
    }
    std::fs::write(&path, content).map_err(|e| e.to_string())
}

#[tauri::command]
async fn load_from_file(path: String) -> Result<String, String> {
    std::fs::read_to_string(&path).map_err(|e| e.to_string())
}
```
